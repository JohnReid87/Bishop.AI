# Bishop.AI

A Windows desktop app for managing AI-assisted coding workflows. A left-hand workspace nav (each workspace bound to a local directory) and a tabbed right pane housing a per-workspace kanban board plus a one-click launcher for the Claude Code Windows app. Designed to grow with additional per-workspace tools (notes, terminal, agent runs) over time.
